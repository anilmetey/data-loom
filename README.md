<h2 id="english">🇬🇧 English Description</h2>

**DataLoom** is a powerful, elegant, and completely client-side data analysis and machine learning platform. Designed with modern Apple-inspired "Bento Grid" aesthetics, it allows you to upload CSV datasets and perform deep data mining directly in your browser—without sending your data to any external server.

### 🌟 Key Features
- **100% Client-Side:** Absolute privacy. Your data never leaves your browser.
- **Modern UI/UX:** Stunning Dark & Light mode integration with responsive, fluid animations.
- **Data Preparation:** Built-in Data Editor to clean missing values (Drop NAs, Fill Mean) and perform Feature Engineering.
- **Statistical Analysis:** View distribution charts, auto-insights, and min/max/mean metrics instantly.
- **Group By Operations:** Aggregate categorical data and visualize the results effortlessly.
- **Linear Regression:** Automatic scatter plots with trendline calculation and Pearson correlation (r) scoring.
- **Machine Learning:** 
  - **K-Means Clustering:** Segment your data into clusters using an integrated custom K-Means algorithm.
  - **Anomaly Detection:** Find statistical outliers using adjustable Z-Score thresholds.
- **Multi-language Support:** Native English and Turkish support.

### 🚀 How to Run
No build process or installation required. Simply open the `index.html` file in any modern web browser to start exploring your data!

<h2 id="türkçe">🇹🇷 Türkçe Açıklama</h2>

**DataLoom**, tamamı tarayıcı üzerinde çalışan, modern ve son derece şık bir veri analizi ile makine öğrenmesi platformudur. Apple ilhamlı "Bento Grid" tasarımıyla geliştirilen bu araç sayesinde CSV formatındaki veri setlerinizi yükleyip, verilerinizin hiçbir sunucuya gitmesine gerek kalmadan doğrudan tarayıcınızda derinlemesine analiz edebilirsiniz.

### 🌟 Öne Çıkan Özellikler
- **%100 İstemci Taraflı (Client-Side):** Tam gizlilik. Verileriniz bilgisayarınızdan dışarı çıkmaz.
- **Modern Arayüz (UI/UX):** Akıcı animasyonlara sahip, göz yormayan Karanlık (Dark) ve Aydınlık (Light) mod desteği.
- **Veri Hazırlama (Prep):** Eksik verileri silme (Drop NA) veya ortalama ile doldurma (Fill Mean) ve yeni değişkenler üretme (Feature Engineering) imkanı.
- **İstatistiksel Analiz:** Dağılım grafikleri, otomatik çıkarımlar (Auto-insights) ve min/max/ortalama değerleri anında görüntüleme.
- **Gruplama (Group By):** Kategorik verileri toplayarak veya ortalamalarını alarak analiz etme.
- **Doğrusal Regresyon (Linear Regression):** Dağılım grafiği (Scatter), trend çizgisi çizimi ve Pearson korelasyon (r) hesaplaması.
- **Makine Öğrenmesi (Machine Learning):**
  - **K-Means Kümeleme:** Kendi içindeki özel algoritması ile verilerinizi farklı kümelere ayırın.
  - **Anomali Tespiti:** Z-Skoru yöntemiyle ekstrem değerleri (outliers) otomatik bulun.
- **Çoklu Dil Desteği:** Türkçe ve İngilizce dillerinde tam destek.

### 🚀 Nasıl Çalıştırılır?
Herhangi bir kuruluma veya backend sunucusuna ihtiyacınız yoktur. Projedeki `index.html` dosyasına çift tıklayıp modern bir web tarayıcısında açmanız yeterlidir. Saniyeler içinde veri madenciliğine başlayabilirsiniz!
